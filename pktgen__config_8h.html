Configuration summary:

This data is used for the startup staged synchronization. It requires the number of islands; for each island, the number of MEs used in the island, and for ach ME the number of contexts used

1 host island 2 Tx islands 1 init island

Host island is two MEs First ME: One thread runs DMA master, others batch distributors Second ME: Runs eight batch distributors

Tx islands is eight MEs (one par batch) All threads run packet receive
